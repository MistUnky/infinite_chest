Infinite Chest
for Minetest

Chests with slots that never run out.

Download GitHub Minetest CoRNeRNoTe

Features

  • Name your chest slots so you can easily find your items.
  • Last accessed slot appears at the top of the list.

Docs

Requirements

  • Minetest >=0.4.3

Installation

Download and uncompress the latest version, and move the the minetest-infinite_chest-master folder into your minetest/mods folder.
